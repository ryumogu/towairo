# Botanical Pet Art - ペットイラストグッズ制作・販売事業 Webサイト

## 📋 プロジェクト概要

「インテリアに溶け込む、うちの子アート」をコンセプトに、ペット写真を「ボタニカル×水彩画風」の繊細なアート作品に変換し、オリジナルグッズとして販売する事業のWebサイトです。

### コンセプト
- **ボタニカル×水彩画風**: 植物の癒しと水彩画の温かみを組み合わせたデザイン
- **アート性重視**: キャラクター的なデフォルメではなく、大人のインテリアに馴染むデザイン
- **初期費用ゼロ**: イラスト制作費は無料で、グッズ購入時に収益化

## 🎯 主な機能

### 実装済み機能

#### 1. トップページ（index.html）
- **ヒーローセクション**: インパクトのあるメインビジュアルとコンセプト訴求
- **コンセプト紹介**: 3つの主要な特徴をビジュアル的に表示
- **制作事例ギャラリー**: 犬・猫・うさぎの Before/After 比較表示
- **CTA（行動喚起）**: 制作依頼への誘導
- **レスポンシブ対応**: スマートフォンでも美しく表示

#### 2. ご利用の流れ（flow.html）
- **5ステップ表示**: 依頼から完成まで視覚的に説明
- **各ステップの詳細情報**: アイコン、説明文、注意事項
- **よくある質問への誘導**: 不安を解消
- **スムーズな導線**: 制作依頼への自然な流れ

#### 3. 注意事項・商品仕様（terms.html）
- **詳細な規約表示**: サービス概要、制作条件、価格設定など
- **視覚的に理解しやすい構成**: アイコンとカラーコーディング
- **同意チェックボックス**: 同意後にのみ依頼フォームへ進める仕組み
- **法的保護**: キャンセルポリシー、免責事項を明記

#### 4. 制作依頼フォーム（request.html）
- **依頼者情報入力**: 名前、メール、電話、その他連絡先
- **ペット情報入力**: ペット名、写真アップロード（1〜3枚）
- **デザイン希望**: 好きな花、植物、色テーマ、その他要望
- **ファイルアップロード機能**: ドラッグ&ドロップ対応、プレビュー表示
- **バリデーション**: 必須項目チェック、メール形式チェック
- **送信完了表示**: 成功メッセージとフィードバック

#### 5. お問い合わせ（contact.html）
- **お問い合わせフォーム**: 名前、メール、件名、内容
- **よくある質問（FAQ）**: アコーディオン形式で主要な質問に回答
- **営業時間・返信目安**: 安心感を提供する情報表示
- **送信完了表示**: 成功メッセージとフィードバック

#### 6. デザイン・UI/UX
- **エレガントな配色**: グリーン系（ボタニカル）とピンク系（優しさ）
- **レスポンシブデザイン**: PC、タブレット、スマートフォン完全対応
- **スムーススクロール**: ページ内リンクのスムーズな移動
- **ハンバーガーメニュー**: モバイル端末でのナビゲーション
- **ホバーエフェクト**: インタラクティブな視覚フィードバック
- **アイコン活用**: Font Awesome による視認性向上

#### 7. JavaScript機能
- **モバイルメニュー**: レスポンシブなハンバーガーメニュー
- **同意チェック**: 利用規約同意後に次ページへ進む制御
- **ファイルアップロード**: 複数画像の選択、プレビュー、削除
- **フォームバリデーション**: リアルタイムの入力チェック
- **FAQアコーディオン**: クリックで開閉するFAQ表示
- **送信処理**: フォーム送信とローディング表示

## 🔧 カスタマイズ方法

### 🔥 期間限定キャンペーンの終了時の変更方法

**無料キャンペーンが終了したら、以下の箇所を変更してください：**

#### 1. トップページ（index.html）の変更箇所

**ヒーローセクションの期間限定バッジ（38行目付近）**
```html
<!-- 🔥 期間限定バッジ（無料期間が終わったらこのdivを削除またはコメントアウト） -->
<div class="limited-badge">
    <i class="fas fa-fire"></i>
    <span>今だけ限定！制作費無料キャンペーン中</span>
</div>
```
→ この `<div class="limited-badge">...</div>` を削除またはコメントアウト

**制作費無料の表示（47行目付近）**
```html
<!-- 🔥 無料期間が終わったら、この「制作費無料」を「制作費5,000円〜」などに変更 -->
<div class="feature-item limited-highlight">
    <i class="fas fa-gift"></i>
    <span>制作費無料</span>
</div>
```
→ 例：`<span>制作費5,000円〜</span>` に変更、`limited-highlight` クラスを削除

**コンセプトカードの内容（100行目付近）**
```html
<!-- 🔥 無料期間が終わったら、h3とpの内容を変更 -->
<!-- 例: <h3>安心の価格</h3> -->
<!-- 例: <p>明確な料金体系で、納得いただいた上で制作を開始します</p> -->
<h3>初期費用ゼロ</h3>
<p><strong class="limited-text">【期間限定】</strong>イラスト制作費は無料。気に入ったらグッズをご購入いただくシステム</p>
```
→ コメントアウトされている例のように変更

**CTAセクションの文言（変更箇所が明示されています）**
```html
<!-- 🔥 無料期間が終わったら、この文を変更 -->
<!-- 例: <p>高品質なボタニカルアートで、大切な思い出を形に</p> -->
<p class="cta-highlight"><span class="limited-badge-small">今だけ！</span>制作費無料・初期費用ゼロで始められます</p>
```
→ コメントアウトされている例のように変更

#### 2. その他のページの変更

- **flow.html**: Step 2の「制作費：無料」の記載を変更
- **terms.html**: サービス概要の「イラスト制作費：無料」の記載を変更
- **contact.html**: FAQの「イラスト制作は本当に無料ですか？」の回答を変更

### 色の変更
`css/style.css` の `:root` セクションでカラー変数を変更：
```css
:root {
    --primary-green: #6B8E6F;  /* お好みの色に変更 */
    --accent-pink: #D4A5A5;
    /* ... */
}
```

### コンテンツの編集
各HTMLファイルを直接編集してください。セマンティックな構造になっているため、編集箇所を見つけやすくなっています。

### 実際の画像の追加
現在はプレースホルダーを使用しています。実際のペット写真とイラストを追加する場合：
1. `images/` フォルダを作成
2. 画像を配置
3. HTMLの `.image-placeholder` 部分を `<img>` タグに置き換え
